## Compiladores:
#### Introducción: Intérpretes vs. Compiladores
En términos muy simplificados, mientras que el **interprete** toma el código, lo ejecuta y lo analiza *en tiempo real*, el **compilador** lo analiza, lo traduce a código de maquina y genera un ejecutable.
#### Proceso de compilación:
###### 1. Inicio:
1. Se toma el código fuente.
###### 2. Análisis:
2. Se hace el Análisis Léxico.
3. Se hace el Análisis Sintáctico.
4. Se hace el Análisis Semántico.
###### 3. Síntesis:
5. Generación de Código Intermedio.
6. Optimización.
7. Generación de Código Final.
###### 4. Final:
8. Se retorna el ejecutable.
